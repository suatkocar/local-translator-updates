# Local Translator

**Privacy-first translation app for macOS** — powered by local AI engines (Ollama & LM Studio) and cloud providers.

## Download

Download the latest version:

**[Download Local Translator](https://github.com/suatkocar/local-translator-updates/releases/latest)**

Look for `LocalTranslator-x.x.x-universal.dmg` in the latest release assets.

## Features

- **Privacy-first**: Run translations entirely on your Mac with local Ollama or LM Studio models — no data leaves your device.
- **Dual local engines**: Full support for both [Ollama](https://ollama.ai) and [LM Studio](https://lmstudio.ai) with automatic CLI installation, model management, and headless server control.
- **Multi-provider**: OpenAI, Anthropic, Gemini, xAI, and OpenRouter support built-in.
- **Global shortcuts**: Translate from any app with customizable keyboard shortcuts.
- **QuickBar**: Spotlight-style floating translation bar for instant access.
- **Screen-to-Translate**: OCR any screen region and translate it instantly.
- **Duel mode**: Compare translations from multiple providers side by side.
- **Translation tones**: 14 tone presets (Professional, Casual, Technical, and more).
- **Diff view**: See word-level changes between original and translated text.
- **48 languages**: Full language support with auto-detection.
- **Menu bar app**: Lives in your menu bar, out of the way until you need it.

## System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac (universal binary)
- For local translation: [Ollama](https://ollama.ai) and/or [LM Studio](https://lmstudio.ai) installed

## Installation

1. Download the `.dmg` file from the [latest release](https://github.com/suatkocar/local-translator-updates/releases/latest).
2. Open the DMG and drag **Local Translator** to your Applications folder.
3. Launch the app — it appears in your menu bar.
4. Configure your preferred AI provider in Settings.

## Automatic Updates

Local Translator checks for updates automatically using Sparkle. You can also check manually from **Settings > Updates**.
